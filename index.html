Trust and permission copy

What the extension reads

The mock extension is framed as reading only the current job posting content you choose to analyze. It does not claim access to unrelated browsing history.

How resume data is handled

Uploaded resumes and job text are processed for the current session and can be deleted by the user at any time.

No selling or sharing

The landing page clearly states that resume data is not sold or shared, reinforcing trust during prototype testing.
